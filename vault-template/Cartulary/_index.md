## Notes

- Reference notes are named `author--title-slug.md`
- The double hyphen `--` separates author from title — reserved only
  here
- If no author is known, use `unknown--title-slug.md`
- A reference note contains a summary of the source plus any
  user commentary, clearly separated
- References are linked *from* Codex notes (via the Glossator);
  they do not usually contain wikilinks themselves
- Book notes from a full reading may be substantial; a web clip note
  is typically brief (summary + URL + user commentary)
